# SkillTree — Визуальный трекер навыков

Превращайте обучение в интерактивную карту. Стройте путь, отмечайте прогресс, делитесь маршрутами.
[Сайт-приложение](https://dinkana.github.io/skilltree/)

## Суть проекта
Стандартные списки задач скрывают структуру предмета. SkillTree визуализирует знания в виде графа. Вы видите взаимосвязи, понимаете, что учить дальше, и получаете мотивацию при разблокировке новых узлов.

## Возможности
* **Наглядная архитектура.** Граф показывает базу и зависимости тем.
* **Игровая механика.** Анимация прогресса и разблокировка узлов удерживают фокус.
* **Установка на смартфон.** Это PWA. Используйте стандартную функцию браузера «Добавить на главный экран» или «Установить приложение». Работает как нативное ПО без магазинов.
* **Полная приватность.** Нет серверов и регистрации. Данные хранятся локально.
* **Мгновенный шеринг.** Обмен деревьями через закодированные ссылки.

## Для кого
* **Студенты и школьники.** Структурирование предметов и подготовка к экзаменам.
* **Специалисты и самоучки.** Построение роадмапов для освоения новых навыков.
* **Наставники.** Создание прозрачных программ для учеников.
