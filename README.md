# Социальный навигатор «Траектория»

PWA-платформа визуальной маршрутизации жизненных и профессиональных навыков для подростков из групп риска.

## Социальная проблема
Подростки из неблагополучных семей и выпускники организаций для детей-сирот не обладают навыками стратегического планирования жизни. Они не видят связи между базовыми социальными действиями и будущим трудоустройством. Традиционные лекции по профориентации не работают из-за отсутствия интерактива и персонализации.

## Целевая группа
Подростки 14–18 лет, находящиеся в трудной жизненной ситуации, воспитанники социальных центров и детских домов. Социальные работники и наставники, сопровождающие подростков.

## Архитектура решения
Социальный тренажер адаптации на базе интерактивного графа. Подросток получает готовый сценарий от наставника. Узлы графа отражают жизненно необходимые компетенции: оформление документов, финансовая грамотность, основы права, поиск рабочего места. Офлайн-режим решает проблему слабого интернета в учреждениях.

## Ключевые функции
* Две роли. Подросток проходит визуальную карту пути. Наставник собирает индивидуальный маршрут из готовых блоков и передает его по локальной ссылке.
* Агрегатор бесплатной помощи. Узлы содержат прямые ссылки на государственные сервисы, инструкции и контакты профильных некоммерческих организаций.
* Офлайн-приоритет. Прогресс сохраняется локально и синхронизируется при появлении сети.
* Безопасность. Отсутствие серверной части и баз данных исключает утечки персональных данных несовершеннолетних.

## Технический стек
* Язык интерфейса: русский.
* Фреймворк: Vue 3, TypeScript, Vite.
* Состояние и стили: Pinia, Tailwind CSS.
* Офлайн-режим: vite-plugin-pwa.
* Хранение данных: LocalStorage.
* Деплой: статический хостинг.

## Ожидаемые социальные результаты
* Формирование навыка целеполагания и понимания алгоритмов взаимодействия с государственными институтами у подростков.
* Снижение уровня тревожности при выходе из организаций для детей-сирот за счет наличия визуального плана действий.
* Создание тиражируемой открытой социальной технологии визуальной маршрутизации для бесплатного использования другими некоммерческими организациями и социальными центрами.
