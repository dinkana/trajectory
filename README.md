# Социальный навигатор «Траектория»

**Грантовое направление:** Поддержка семьи, материнства, отцовства и детства  
**Тематика:** Постинтернатное сопровождение молодых людей из числа детей-сирот и детей, оставшихся без попечения родителей. Социальная адаптация детей-сирот, подготовка их к самостоятельной взрослой жизни.

## Социальная проблема и обоснование
Выпускники организаций для детей-сирот и подростки из групп риска не обладают навыками стратегического планирования жизни. Они не видят связи между базовыми социальными действиями и будущим трудоустройством, что приводит к росту тревожности и социальной изоляции при выходе из учреждений. Традиционные лекции по профориентации и социальной адаптации неэффективны из-за отсутствия интерактива, персонализации и доступности в офлайн-среде.

## Целевые группы
**Основная:** Подростки 14–18 лет, находящиеся в трудной жизненной ситуации, воспитанники социальных центров и детских домов.  
**Вторичная:** Социальные работники, наставники и педагоги, сопровождающие подростков.

## Механизм реализации
Проект представляет собой PWA-платформу визуальной маршрутизации жизненных и профессиональных навыков. 
1. **Роль наставника:** Социальный работник использует веб-интерфейс для сборки индивидуальной программы социального сопровождения из готовых верифицированных блоков (пресетов) и передает маршрут подопечному через QR-код или локальную ссылку.
2. **Роль подопечного:** Подросток открывает приложение по ссылке, видит визуальную карту пути и последовательно осваивает этапы. Закрытые узлы требуют выполнения предыдущих шагов.
3. **Интеграция ресурсов:** Каждый узел графа содержит не просто текст, а прямые ссылки на бесплатные государственные сервисы (Госуслуги, МФЦ), видео-инструкции от НКО и шаблоны заявлений.
4. **Офлайн-приоритет:** PWA-оболочка позволяет загрузить трек в учреждении и проходить этапы дома, где связь нестабильна. Прогресс синхронизируется локально.

## Ожидаемые социальные результаты
**Количественные:**
- Количество подростков, составивших и прошедших индивидуальную траекторию адаптации.
- Количество социальных работников, внедривших инструмент в практику сопровождения.
- Охват географии пилотных регионов и социальных учреждений.

**Качественные:**
- Формирование у подростков навыка целеполагания и понимания алгоритмов взаимодействия с государственными институтами.
- Снижение уровня тревожности при выходе из организаций для детей-сирот за счет наличия визуального плана действий.
- Создание тиражируемой открытой социальной технологии визуальной маршрутизации.

## Устойчивость и тиражируемость
Технология является полностью открытой и не требует серверной инфраструктуры (статический PWA-хостинг). Отсутствие серверной части и баз данных исключает затраты на поддержку после завершения гранта и гарантирует абсолютную безопасность персональных данных несовершеннолетних. Это позволяет масштабировать практику на социальные центры и НКО по всей стране без дополнительных финансовых вложений.

## Технический стек
- Фреймворк: Vue 3, TypeScript, Vite
- Состояние и стили: Pinia, Tailwind CSS
- Офлайн-режим: vite-plugin-pwa (Workbox)
- Хранение данных: LocalStorage
- Деплой: Статический хостинг (GitHub Pages)
